## Project Overview
The **Automatidata Project** is an end-to-end data analytics and machine learning project completed as part of the **Google Advanced Data Analytics Professional Certificate**.  
It simulates a real-world business scenario where data analysts work with stakeholders to analyze data, perform statistical inference, and build predictive models.

The project uses a **New York City Taxi Trip dataset** to derive insights and develop regression and classification models.

---

## Project Objectives
- **Fare Prediction (Regression)**
  - Predict taxi fare amounts based on trip features
  - Identify key factors influencing fare pricing

- **Tip Prediction (Classification)**
  - Classify trips into high-tip vs low-tip categories
  - Understand passenger tipping behavior

---

## Dataset Description
The dataset is a pedagogical version of the NYC Taxi dataset and includes:
- Trip distance
- Trip duration
- Passenger count
- Pickup and drop-off locations
- Fare amount
- Tip amount
- Payment type

The data reflects real-world challenges such as skewed distributions, outliers, and correlated features.

## Key Insights
- Trip distance and duration are strong predictors of fare amount
- Multicollinearity impacts coefficient stability in linear regression
- Tree-based models perform well for tip classification
- Feature engineering improves overall model performance
